## 使用方法

```bash
# 在某个强趋势时段上测试策略
af run my_strategy.py --symbols J --start 2016-03-24 --end 2016-11-09 --freq daily

# 在多个时段上验证策略鲁棒性
af run my_strategy.py --symbols ZC --start 2021-03-01 --end 2021-10-31
af run my_strategy.py --symbols I --start 2016-06-01 --end 2017-01-31
af run my_strategy.py --symbols SA --start 2023-06-01 --end 2024-01-31
```
